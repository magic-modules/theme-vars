@magic-modules/theme-vars

this is the @magic-modules ThemeVars component.

embed a list of the theme vars on a @magic-theme demo page.

installation

npm install @magic-modules/theme-vars

usage

first, copy the THEME_VARS from the theme/index.mjs file to the /src/app.mjs file

/src/app.mjsexport const state = {  // ...other state,  vars: {    background: {      dark: '#232323',      light: '#eeeeee',    },    text: {      dark: '#cccccc',      light: '#232323',    },    link: {      dark: '#eeeeee',      light: '#010101',      hover: {        dark: '#fefefe',        light: '#010101',      },    },    fadeDuration: '300ms',    widths: {      laptop: '1000px',    },  },}

then, in a page:

<ThemeVars state></ThemeVars>

demo

theme vars

colors

this theme exports and uses the following colors.

  • background
    • vars.background.light: #eeeeee
    • vars.background.dark: #232323
  • link
    • vars.link.light: #010101
    • vars.link.dark: #eeeeee
    • link.hover
      • vars.link.hover.light: #010101
      • vars.link.hover.dark: #fefefe
  • text
    • vars.text.light: #232323
    • vars.text.dark: #cccccc

fade duration:

vars.fadeDuration should be used for all css animations

vars.fadeDuration: 300ms

breakpoints

the following width breakpoints exist

  • vars.widths.laptop: 1000px
