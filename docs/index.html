@magic-modules/theme-vars

this is the @magic-modules ThemeVars component. embed a list of the theme vars on a @magic-theme demo page.

installation:

npm install @magic-modules/theme-vars

usage:

in a page or module View, after copying the vars from the theme/index.mjs file to the /app.mjs file

/app.mjsexport const state = {  // ...other state,  vars: {    background: {      dark: '#232323',      light: '#eeeeee',    },    text: {      dark: '#cccccc',      light: '#232323',    },    link: {      dark: '#eeeeee',      light: '#010101',      hover: {        dark: '#fefefe',        light: '#010101',      },    },    fadeDuration: '300ms',    widths: {      laptop: '1000px',    },  },}
/pages/index.mjsexport const View = (state) => ThemeVars(state.vars)

demo

theme vars

colors

@magic-themes/docs exports and uses the following colors.

  • background
    • vars.background.dark: #232323
    • vars.background.light: #eeeeee
  • link
    • vars.link.dark: #eeeeee
    • vars.link.light: #010101
    • link.hover
      • vars.link.hover.dark: #fefefe
      • vars.link.hover.light: #010101
  • text
    • vars.text.dark: #cccccc
    • vars.text.light: #232323

fade duration:

vars.fadeDuration should be used for all css animations

vars.fadeDuration: 300ms

breakpoints

the following width breakpoints exist

  • vars.widths.laptop: 1000px
